Développement de l'application Idea Drop avec le framework TanStack, Vite et React.
L'API pour ce projet est disponible ici('https://github.com/figura01/idea-drop-api')

# Fonctionnalitées

- Créer un compte
- Login/logout
- Créer et déposer une idée
- Editer ou supprimer ses idées
- Consulter les différentes idées
- Gestion des sessions avec JWT

# Technos

- Vite
- TanStack Router
- TanStack Query
- React
- Axios
- Tailwind
- Typescript
